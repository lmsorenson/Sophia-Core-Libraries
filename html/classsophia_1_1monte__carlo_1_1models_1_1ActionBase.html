Detailed Description

template<typename TState, typename TChange>
class sophia::monte_carlo::models::ActionBase< TState, TChange >

A template base class for implementing the Action interface.

This class provides a generic implementation of an action, using templates to allow for different types of states and changes. It handles the connection between a source and a target node.

Template Parameters
TStateThe type representing the state of the system (e.g., a board configuration).
TChangeThe type representing the change that this action applies to a state (e.g., a move).
