A template base class for implementing the {\ttfamily \doxylink{classsophia_1_1monte__carlo_1_1models_1_1Node}{Node}} interface. 

This class provides a generic implementation of a node, using templates to support various types of states and changes. It manages the state of the node and provides access to a factory for creating related objects in the tree.
