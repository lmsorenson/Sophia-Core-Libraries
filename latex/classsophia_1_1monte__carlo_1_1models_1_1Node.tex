\doxysubsection{Detailed Description}
Represents a node in the Monte Carlo Search Tree. Each node corresponds to a state in the problem space. 

The \doxylink{classsophia_1_1monte__carlo_1_1models_1_1Node}{Node} class is central to the MCTS algorithm. It stores statistics about the state it represents, such as the number of visits and the total reward. It also provides methods for the four main phases of MCTS\+: selection, expansion, simulation (rollout), and backpropagation.
