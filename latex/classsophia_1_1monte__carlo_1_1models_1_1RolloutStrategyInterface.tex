\doxysubsection{Detailed Description}
Interface for defining a rollout strategy in the Monte Carlo Tree Search (MCTS) algorithm. 

A rollout (or simulation) is a traversal from a given node to a terminal state of the problem space. The rollout strategy defines how actions are selected during this simulation phase. Different strategies can be implemented to balance exploration and exploitation.
