\doxysubsection{Detailed Description}
Represents an action that can be taken from a specific state (\doxylink{classsophia_1_1monte__carlo_1_1models_1_1Node}{Node}) in the search tree. 

An action is a transition from a source node to a target node. It represents a move, a decision, or any other change in the state of the system being modeled.
