\subsubsection*{template$<$typename TState, typename TChange$>$\newline
class sophia\+::monte\+\_\+carlo\+::models\+::\+Action\+Base$<$ TState, TChange $>$}
A template base class for implementing the {\ttfamily \doxylink{classsophia_1_1monte__carlo_1_1models_1_1Action}{Action}} interface. 

This class provides a generic implementation of an action, using templates to allow for different types of states and changes. It handles the connection between a source and a target node.


\begin{DoxyTemplParams}{Template Parameters}
{\em TState} & The type representing the state of the system (e.\+g., a board configuration). \\
\hline
{\em TChange} & The type representing the change that this action applies to a state (e.\+g., a move). \\
\hline
